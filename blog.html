1 | Discuss the scope of var , let , and const ?

Var : Variables declared with var are function scoped. This type of variables only accessible in where they are declared. Var variables also hoisted to the top of their function,this means I can use a var variable before it's declared.
Let : Variables declared with var are block scoped.They are only accessible within the nearest enclosing block (if, for, or while statement) where they are defined. Let variables are not hoisted to the top of their scope.That's why i cann't use let variables before they are declared.
Const : Const variables are block scoped.Like let variables they are not hoisted to the top of their scope.Const variables are used to declare constant.Which means the value i assigned to the const variables cannot be changed or reassigned.

2 | Tell us the use cases of null and undefined ?

Undefined : In JS ,if I declared a variables but not initializes its value then its automatically assigned the value undefined.If a function is called with less arguments than it expects,missing parameters are set to undefined also if I return nothing from a function it return undefined
